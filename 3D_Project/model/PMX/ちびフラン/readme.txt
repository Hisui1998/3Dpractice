■ポイント
座らせる場合スカートの物理を切ってください。

顔には影があまり落ちないよう設定してあります。シェーダを使う際は
気をつけてください。

一般のモデルと頭身が異なりますのでモーション流し込みの際は
ボーン位置角度補正のボーン位置に適宜、値を入力して補正してください。
だいたい*0.6〜0.8あたりでそれっぽくなります。

sm27099243←こちらの動画にて配布されているプラグインにて、モデルに
特殊な構造を施すことでモーション流し込みの際の頭身の違いによるズレ
をかなり軽減できるみたいです!

羽の宝石はAutoLuminous対応です。光らせたくない場合はpmxエディターを
使い、材質「宝石」の反射強度を99以下にしてください。
